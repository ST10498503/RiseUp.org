## Changelog

### Part 3 Updates
- Added JavaScript search functionality on Programs page.
- Implemented sorting of programs (A-Z and Z-A).
- Added dynamic program listings using JavaScript DOM manipulation.
- Implemented jQuery accordion on Programs page.
- Added tabbed interface for program categories.
- Implemented image gallery with lightbox functionality.
- Added AJAX form submission on Contact page.
- Added JavaScript validation on Contact and Enquiry forms.
- Added interactive map using Leaflet.
- Added scroll animations using CSS and JavaScript.
- Improved website responsiveness and UI design.
- Added robots.txt and sitemap.xml files.
- Added SEO meta tags on all pages.

Security Measures Implemented:
- Client-side validation.
- Input sanitisation using JavaScript trim().
- HTML5 validation.
- HTTPS deployment through Netlify.
